---- Sobre o Projeto

O projeto é desenvolvido durante o período letivo do curso de Ciência da Computação Na FEI-SBC, o objetivo principal é desenvolver uma carteira digital no qual dois ou mais usuários podem efetuar operações semelhantes aos bancos virtuais - ações de compra, venda e consulta de criptomoeda, consulta de extrato, etc - , assim, explorando as capacidades técnicas utilizando o máximo para aperfeiçoar capacidades de programação e o uso que a linguagem C disponibiliza
